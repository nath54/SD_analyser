Stable Diffusion Analyser

by nath54

This website is designed to evaluate Stable Diffusion open source models (and more generally, all the good txt2img models I can run on a local computer) on different test prompts to compare the models between them.

There are two different test sets : the knowledge ones and the style/attribute ones. The knowledge tests indicate from a library of entities/concepts the elements the different models perfectly generate; generate with some errors; or can't generate at all.

On the other side, the style/attribute tests shows how the models react with different style prompt expressions or attribute prompt expressions combination.

How was the experiment made?

For the moment, I generate everything on my local laptop (rtx 3060 - 6gb of VRAM), the pictures have the dimension 768x768. I generate them all in 35 steps using the sampler DKMS++ 2M SDE Karras, with each picture its own random seed. I also use common negatives prompts.

If you want more details about the prompts, you will get them by clicking the info box of each pictures to see them prompts.

It tooks about 5~10 seconds to generate one picture on my laptop, so you guess it takes a long time to generate everything.
